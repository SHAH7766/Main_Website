# Agency Website (Vite + React)

Minimal starter for an agency landing site (digital marketing, AI automation, web design).

Quick start (Windows):

```powershell
cd "c:\Users\shahu\Desktop\USA_Website\Main_Website"
npm install
npm run dev
```

Open http://localhost:5173

Next steps:
- Customize content and colors in `src`.
- Hook the contact form to your email/CRM.
- Add images and analytics.
Tailwind setup:

```powershell
npm install -D tailwindcss postcss autoprefixer @tailwindcss/typography
npx tailwindcss init -p
```

Then run `npm run dev`.
